New EC2 Installation guid: 
    Get permisson for docker command: sudo chmod 666 /var/run/docker.sock 
    Get Permission for docker-compose command: chmod +x ~/.docker/cli-plugins/docker-compose
    Install docker-compose: sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose [source: https://www.digitalocean.com/community/tutorials/how-to-install-and-use-docker-compose-on-ubuntu-22-04]
